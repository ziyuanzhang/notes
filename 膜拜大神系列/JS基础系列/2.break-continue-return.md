## break 与 continue 与 return

break ：跳出循环/代码块；  
1.循环或者 switch 中；2. 带标签的“循环”代码块中使用；3.带标签的“非循环”代码中使用

```
    //跳出标签foo所在的循环/代码块
    foo:for (let i = 0; i < 4; i++) {
            for (let j = 0; j < 4; j++) {
                if (i * j >= 3) {
                    console.log("stop:", i, j)
                    break foo
                }
                console.log(i, j)
            }
        }
    // 0 0
    // 0 1
    // 0 2
    // 0 3
    // 1 0
    // 1 1
    // 1 2
    // stop:1 3
```

```
    //bar为标签；跳出代码块
    function foo() {
        bar: {
            console.log("hello");
            break bar;
            console.log("world")
        }
    }
    foo()；
    // hello
    // world
```

continue ：跳过本次循环继续执行  
1.循环或者 switch 中；2. 带标签的“循环”代码块中使用；

```
   //continue执行foo循环的下一轮循环
    for (let i = 0; i < 4; i++) {
        for (let j = 0; j < 4; j++) {
            if (i == j) {
                continue foo;
            }
            console.log(i, j)
        }
    }
    //1 0
    //2 0
    //2 1
    //3 0
    //3 1
    //3 2
```
