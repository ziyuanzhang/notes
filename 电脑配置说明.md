- 轻薄笔记本：优势就是薄和轻；Office、 线上会议、 影音娱乐，玩玩 LOL，DNF、CF 等这类不很吃性能的游戏，续航能力好、静音，

- 全能本：配置低功耗显卡，能有不错的图形性能，日常办公兼顾游戏是不错的选择；2023 年仅推荐 RTX2050、RTX3050（4GB）、RTX3050（6GB）显卡，可以应付普通设计、视频剪辑需求。一般只适合中轻度游戏玩家使用
  全能本、商务本、设计师本之类的不同品牌厂家各自定义不同，因此会有表达差别，不过是一般也会以“高性能轻薄本”统称指代。
  一旦涉及视频创作、3D 建模、游戏、大量剪辑视频、需求这类图形性能要求高的应用；
  全能本因为有独立显卡的加持，平面设计、视频剪辑、动画建模、特效渲染等生产力工作表现要高于采用核显的常规轻薄本，更加适合商务使用；一般只适合中轻度游戏玩家使用

- 轻薄游戏本：介入全能本和游戏本中间，拥有不错的性能，重量一般在 2.0kg，不管是游戏、设计都是不错的选择，适合需要一定性能以及便携的用户。

- 游戏本也叫高性能笔记本，如果平常喜欢玩 2077、古墓丽影、GTA5、大表哥等 3A 大作游戏的话一定要选择性能强的游戏本才能在高特效下流畅运行（3A 玩的是画质），毕竟笔记本不能和台式机一样可以更换显卡、CPU 等这些配件进行升级；而且一般笔记本的寿命在五年左右，其配置就会变落后；同样专业需要 PS、绘图、视频剪辑、AE、CAD 建模、渲染等这些设计软件也是需要性能好的笔记本才能多开和流畅运行。因为强性能，需要更好的散热模组，所以其机身也会笨重些，比较适合定点使用。

分辨率：2K 够用

## CPU: ---主要影响电脑的流畅程度和反应快慢

系列 + 代数 + 数字 + 后缀 （例如 i7-12900H: i7 系列 + 12 代 + 900 + H）

**注：** 看后三位数字，数字越大，CPU 性能越强；英特尔强于 AMD

13 代酷睿移动端分为 HX，H，P，U 四个系列。

1. HX 系列: TDP 是 55W，用在游戏本； 高性能游戏本；（桌面端的搬迁到移动端上，都支持超频和内存超频。）
2. H 系列【标准电压】: TDP 是 45W，全能本用；普通游戏本或者全能本；（相比 12 代：性能没提升，提升续航）
3. P 系列【准标压】: 是 28W，全能本和高性能轻薄本；（阉割比较严重的：四千出头可以考虑，太贵 AMD）
4. U 系列【低电压】: 是 15W，轻薄本或者超级本用。（阉割比较严重的：三千多可以考虑，太贵 AMD）

从性价比角度，肯定推荐 H45 的 cpu 了，但 H45 不一定就很好用，P 系列其实更适合商务办公，U15 也是可以用的，如果你给他足够的功耗，性能也不差，但如果笔记本功耗真的仅仅 15W，那么长期来看，保值角度确实不太好；

## 显卡（GPU）：独显 / 核显 /集显(被淘汰了) -- 主要影响电脑的图像性能，比如 Pr 预览的流畅性、导出视频的快慢、3D 软件渲染的速度、游戏的帧率等方面

- 独显:

  1. NVIDIA--RTX 独显: 高性能独显，可以用来打游戏、做设计
  2. NVIDIA--GTX 独显【阵亡】: 性能弱于 RTX 独显（前面的图），目前在售的 GTX 独显本以全能本居多
  3. NVIDIA--MX 独显【2023 阵亡】: 入门级独显，性能稍强于核显，搭载此类独显的多为轻薄本
  4. AMD 独显：因为驱动程序问题多，不建议小白入手。

  RTX 显卡和 MX 系列显卡，显卡主要看数字，数字越大，性能越强

- 核显: AMD 核显还是可以考虑一下的，R7 的核显强于 R5 的核显。AMD 的 RX 680M 核显（R7 和 R9 处理器上使用的核显）的性能接近 NVIDIA 的入门级 MX 独显，平时玩玩网游绰绰有余了

显卡最重要的是功耗，W 是功耗的单位，显卡的功耗不易看出，但往往功耗高的显卡都会在商店界面标出来，生怕你不知道他是满血卡。（如果支支吾吾的，那么显卡的功耗一定低的可怜）

## 内存 : 16G 起步，双插槽内存（可加装内存条）

## 硬盘 : 512GB 以上（Windows11 占将近 100G），固态硬盘性能(SSD) > 机械硬盘(HDD)

## 屏幕 : “高刷屏”与“高分屏”要 协同

1. 宽高：16:10;
2. 高刷屏：60Hz 以上为高刷新率，游戏本起步得是 144Hz;
3. 高分屏：1920*1080【1080P】、2560*1440【1440P / 2K】、3840\*2160【4K】;

DC 调光和莱茵低蓝光护眼认证，超过 6 小时 需要蓝光，不过影响不大

## 接口 : 至少要有 C 口、USB 接口、电源接口和 3.5mm 的耳麦接口

## 散热:无

## 键盘和触摸板:无

## 音响:无
