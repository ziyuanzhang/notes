## 1. Python Web 后端开发（FastAPI/Django/Flask）

Python → Web开发 → 数据库 → 网络 → Linux → 缓存 → 消息队列 → 微服务 → 云原生

- Python后端全景图

  ```bash
    业务代码
    │
    ├─ Python
    │
    ├─ FastAPI
    │
    ├─ SQLAlchemy
    │
    ├─ Redis
    │
    ├─ RabbitMQ/Kafka
    │
    ├─ MySQL
    │
    ├─ HTTP/TCP/IP
    │
    ├─ Linux
    │
    ├─ Docker
    │
    ├─ CI/CD
    │
    └─ Kubernetes
  ```

- 按就业和实际开发优先级排序

  ```bash
    第一优先级（必须）
    Python
    ↓
    MySQL
    ↓
    Linux
    ↓
    HTTP/TCP/IP
    ↓
    FastAPI
    ↓
    Redis

    第二优先级（进阶）
    asyncio
    ↓
    pytest
    ↓
    Docker

    第三优先级（高级）
    RabbitMQ/Kafka

  ```
