## uv 安装与使用

1. 安装
   - macOS / Linux: `curl -LsSf https://astral.sh/uv/install.sh | sh`
   - Windows (PowerShell): `powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"`
   - 备选安装方式
     - macOS (Homebrew): `brew install uv`
     - Windows (Winget): `winget install uv`
     - 通过 pipx 隔离安装: `pipx install uv`

2. 验证安装：`uv --version`
3. 初始化新项目：`uv init my_project`; `cd my_project`
4. 管理 Python 版本:
   - 安装指定的 Python 版本：`uv python install 3.12`
   - 锁定当前项目的 Python 版本：`uv python pin 3.12`（会生成或更新 .python-version 文件）

5. 管理依赖包
   - 添加依赖：使用 uv add 安装包，并自动将其记录到 pyproject.toml 中 `uv add requests`。
   - 添加开发依赖（如测试工具）：`uv add --dev pytest`

6. 同步项目环境: `uv sync`
   - 当你在团队协作中拉取了包含 pyproject.toml 和 uv.lock 的项目时，只需一键同步，uv 会自动创建虚拟环境并安装所有依赖：

7. 运行代码（最省心的命令）: `uv run python main.py`

- 使用 uv run 是最推荐的运行方式。它会自动在项目的虚拟环境中执行代码，完全不需要你手动去 source activate 激活环境：

## 安装并初始化项目

1. 初始化项目: `uv init my_project` ；`cd my_project`;
2. 创建虚拟环境并激活：
   - `uv venv`;
   - Linux/macOS: `source .venv/bin/activate`
   - Windows (PowerShell/CMD): `.venv\Scripts\activate`
3. 添加”.python-version “文件，在src下写代码；
4. git:
   - `git add uv.lock`
   - .gitignore（最小但正确）

   ```gitignore
     .venv/
     __pycache__/
     .mypy_cache/
     .pytest_cache/
     .ruff_cache/
     .env
   ```

5. 安装开发依赖:`uv add --dev pytest mypy ruff`
   在 pyproject.toml 中添加：

   ```python
      [tool.mypy]
      python_version = "3.11"
      strict = true

      # 项目结构
      packages = ["my_project"]
      mypy_path = ["src"]

      # 常见妥协
      ignore_missing_imports = true
      warn_unused_ignores = true
      warn_return_any = true
      warn_unreachable = true

      [tool.ruff]  # 格式 + lint
      target-version = "py311"
      line-length = 88

      [tool.ruff.lint] # 格式 + lint
      select = ["E", "F", "I", "B", "UP"]
      ignore = ["E501"]
   ```

   - 执行格式检查和格式化：`uv run ruff check .`；`uv run ruff format .`；
   - 执行类型检查：`uv run mypy src`；

6. 同步下载pyproject.toml中的依赖：`uv sync`
   👉 uv + mypy + ruff + src 布局
   👉 FastAPI + uv + mypy + ruff 标准骨架
   👉 LangGraph / LlamaIndex 项目如何用 uv 管理多子模块
