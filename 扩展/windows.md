# windows

## win10、11 修改 “微软应用商店” 软件安装位置

## win10 开启 WSL

- 开启 WSL

  1. 左下角 win 图标，右键“应用与功能”
  2. 右上角“程序和功能”
  3. 启用或关闭 Windows 功能
  4. 下滑，找“适用于 linux 的 Windows 子系统”，勾选
  5. 重启

- 安装 ubuntu

  1. 微软应用商城, 搜“ubuntu”，获取
  2. 启动；

- 切换 ubuntu 的终端

  1. 微软应用商城，搜“Windows terminal”，获取；
  2. 启动（默认 Windows 终端）；界面: 下拉菜单--》选择 ubuntu；
  3. 设置默认 Ubuntu：界面: 下拉菜单：设置--》启动--》默认配置文件，选 Ubuntu；
  4. 保存
