# AI 大纲

AI算法工程师 、AI应用工程师

## AI算法工程师

- 学习内容：

  ```code
      数学
      线性代数
      概率统计
      机器学习
      深度学习
      模型训练
      模型微调
  ```

- 技术：PyTorch、TensorFlow

## AI应用工程师

学习内容：

```bash
    Python
    FastAPI
    数据库
    Redis
    Docker
    LLM
    RAG
    Agent
```

### 第一阶段：把后端打牢

- FastAPI进阶

  重点：
  1. Depends
  2. Middleware
  3. JWT
  4. RBAC
  5. WebSocket
  6. BackgroundTasks

- SQLAlchemy

  必须掌握：
  1. Session
  2. ORM
  3. Relationship
  4. 事务
  5. 分页

- Redis

  掌握：
  1. 缓存
  2. 分布式锁
  3. 消息队列
  4. 限流

- Docker
  掌握：
  1. 镜像
  2. 容器
  3. Docker Compose

### 第二阶段：LLM基础

- Prompt Engineering
  1. Role
  2. Task
  3. Constraint
  4. Output Format

- Function Calling

  LLM -> 决定调用工具 -> 执行工具 -> 返回结果

- Embedding

  文本 -> 向量 -> 相似度搜索

### 第三阶段：RAG

- Retrieval
- Augmented
- Generation

推荐：Qdrant 简单且流行。

### 第四阶段：Agent

### 第五阶段：AI系统架构

典型架构：

```bash
    React/Vue
    ↓ HTTP
    FastAPI
    ├── 用户系统
    ├── 会话管理
    ├── Agent
    ├── RAG
    ├── 文件上传
    ├── 权限控制
    ↓
    Redis
    ↓
    MySQL
    ↓
    Qdrant
    ↓
    LLM
```
