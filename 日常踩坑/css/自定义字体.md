# 自定义字体

```css
@font-face {
  font-family: "CiscoSansBold";
  src: url("../fontface/ciscosansbold-webfont.eot"); /* IE9 compat Mode */
  src: url("../fontface/ciscosansbold-webfont.eot?#iefix") format("embedded-opentype"),
    /* Ie6-Ie8 */ url("../fontface/ciscosansbold-webfont.woff") format("woff"),
    /* Modern Browser */ url("../fontface/ciscosansbold-webfont.ttf") format("truetype"),
    /* Safari, Android, iOS */
      url("../fontface/iscosansbold-webfont.svg#CiscoSansBold") format("svg"); /* Legacy iOS */
  font-weight: normal;
  font-style: normal;
}

@font-face {
  font-family: "CiscoSansHeavy";
  src: url("../fontface/ciscosansheavy-webfont.eot");
  src: url("../fontface/ciscosansheavy-webfont.eot?#iefix") format("embedded-opentype"),
    url("../fontface/ciscosansheavy-webfont.woff") format("woff"),
    url("../fontface/ciscosansheavy-webfont.ttf") format("truetype"), url("../fontface/ciscosansheavy-webfont.svg#CiscoSansHeavy")
      format("svg");
  font-weight: normal;
  font-style: normal;
}
```
