# 字母大小写转化

|      属性      | 值         | 说明                    |
| :------------: | ---------- | ----------------------- |
| text-transform | none       | 默认                    |
|       --       | capitalize | 每个单词 以大写字母开头 |
|       --       | uppercase  | 所有字母大写            |
|       --       | lowercase  | 所有字母小写            |
|       --       | inherit    | 继承                    |
